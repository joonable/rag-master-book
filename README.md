# RAG Master Book 실습 가이드

이 저장소는 **'RAG Master'** 책의 튜토리얼을 실습하고, **Gemini CLI**를 활용하여 검색 증강 생성(RAG) 기술을 익히기 위한 프로젝트입니다.

## 🚀 프로젝트 시작하기

본 프로젝트는 의존성 관리를 위해 [uv](https://github.com/astral-sh/uv)를 사용합니다.

```bash
# 가상환경 생성 및 활성화
uv venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate

# 패키지 설치
uv sync
```

## 📖 학습 목차

### [Chapter 1] 랭체인 살펴보기
- **1 랭체인 개요:** 주요 패키지, 버전별 기능, 활용 사례
- **2 대규모 언어 모델:** 파라미터 설정, 주요 모델 활용
- **3 랭체인 표현 언어(LCEL):** Runnable 표준 인터페이스, 체인 연결
- **4 프롬프트:** 퓨샷 프롬프트, 프롬프트 허브
- **5 출력 파서:** Pydantic, JSON 파서 활용
- **6 메모리 관리:** 대화 기록 유지, 요약 및 트리밍

### [Chapter 2] 검색 증강 생성 기초와 실습
- **1 RAG 개요:** 텍스트 임베딩, 코사인 유사도, 랭체인 임베딩 API
- **2 문서 로더:** 웹, PDF, CSV 로더
- **3 텍스트 분할:** 재귀적 문자 분할, 시맨틱 청킹
- **4 벡터 데이터베이스:** Chroma, FAISS
- **5 RAG 챗봇 실습:** 챗봇 구현 및 Streamlit UI 적용

### [Chapter 3] 멀티모달 RAG를 활용한 복합 데이터 처리
- **1 멀티모달 RAG 개요:** 개념 및 도전 과제
- **2 구현 방법:** 벡터 공간 통합, 기본 모달리티 표현, 별도 저장소 관리
- **3 실습:** 멀티-벡터 검색기 및 멀티모달 RAG 구현

### [Chapter 4] 검색과 응답을 최적화하는 RAG 고도화 전략
- **1 청킹 전략:** 부모-자식 분할
- **2 질의 변형:** 다중 질의 생성(Multi-query), 가상 문서 임베딩(HyDE)
- **3 검색 알고리즘:** 희소/밀집/앙상블 검색
- **4 문서 후처리:** LLM 기반 리랭킹, 크로스 인코더
- **5 확장된 방법론:** Self-RAG 개요 및 구현

### [Chapter 5] 지식 그래프를 활용한 그래프 RAG
- **1 그래프 RAG 개요:** 지식 그래프 동작 과정
- **2 그래프 DB 구축:** 데이터 모델링 및 구축 실습
- **3 그래프 RAG 질의:** 질의 과정 및 실습
- **4 통합:** Neo4j와 랭체인 연동

### [Chapter 6] 랭그래프로 설계하는 RAG 파이프라인
- **1 랭그래프 구성요소:** 그래프, 상태, 노드, 에지
- **2 활용:** 루프, 조건문, 스트리밍, 상태 저장, 개입
- **3 실습:** 자체교정-RAG(Self-Corrective RAG), 코드 어시스트 챗봇

### [Chapter 7] 리액트 에이전트를 활용한 RAG
- **1 생각의 사슬 (CoT)**
- **2 에이전트 RAG:** 도구 제작, 프롬프트 설정, 객체 생성 및 실습

### [Chapter 8] RAG 성능을 높이는 LLM 파인튜닝
- **1 RAFT 논문:** 네거티브 샘플, 생각의 사슬
- **2 성능 향상 팁:** 답변 없음 처리, 출처 인용
- **3 데이터셋:** 학습 데이터 탐색 및 전처리
- **4 Qwen 파인튜닝:** RunPod 환경, LoRA 학습, 모델 테스트

### [Chapter 9] 임베딩 모델 파인튜닝
- **1 학습 원리:** 대조 학습, 배치 내 네거티브 샘플링, MNR Loss
- **2 성능 최적화:** 배치 크기, 하드 네거티브 선정
- **3 실전 파인튜닝:** 합성 데이터 생성, 모델 학습 및 성능 평가(mAP, Hit Rate 등)
